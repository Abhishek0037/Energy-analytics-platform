#  Energy Analytics Platform

An interactive web application built using Python, Streamlit, Pandas, and Plotly to analyze energy consumption data, estimate electricity costs, and generate insights from uploaded CSV datasets.

## Features

- Upload energy datasets (CSV)
- Dynamic column mapping
- Energy consumption KPIs
- Consumption trend visualization
- Monthly cost estimation
- Monthly cost breakdown
- Automated insights
- Downloadable reports

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
